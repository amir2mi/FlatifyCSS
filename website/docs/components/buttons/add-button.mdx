By adding `.add-button` to `<button>` it will be a button with a plus sign, you can use this component for menus or other interactive elements. `.active` class will change plus to minus.

:::info Accessibility matters!
You do not need to add anything inside the add button element but it is necessary to set an `aria-label` HTML attribute for accessibility purposes. [Read more about aria-label usage](https://developer.mozilla.org/en-US/docs/Web/Accessibility/ARIA/ARIA_Techniques/Using_the_aria-label_attribute).
:::

<PreviewBox
  height="70px"
  children='
<!DOCTYPE html>
<html lang="en">
	<head>
		<meta charset="UTF-8" />
		<meta http-equiv="X-UA-Compatible" content="IE=edge" />
		<meta name="viewport" content="width=device-width, initial-scale=1.0" />
		<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/flatifycss/dist/css/flatify.min.css" />
		<title>Add button</title>
		<style>
			.add-button{
				margin: 0 5px !important;
			}
		</style>
	</head>
	<body>
        <div class="flex-center">
			<button class="add-button interactive" aria-label="Add"/></button>
			<button class="add-button" aria-label="Add"/></button>
			<button class="add-button active" aria-label="Remove"/></button>
		</div>
	</body>
	<script>
		var addButton = document.querySelector(".add-button.interactive");
		addButton.addEventListener("click", () => {
			addButton.classList.toggle("active");
			if(addButton.classList.contains("active")){
				addButton.setAttribute("aria-label", "Remove");
			}else{
				addButton.setAttribute("aria-label", "Add");
			}
		});
	</script>
</html>
'
/>

```html title="index.html"
<button class="add-button" aria-label="Add"/></button>
<button class="add-button active" aria-label="Remove"/></button>
```

### Sizes

To change the button size use [size setter classes](/docs/helpers/size-setters).
These classes just set a `font-size` property so it is possible to change it yourself by writing custom CSS.

<PreviewBox
  height="130px"
  children='
<!DOCTYPE html>
<html lang="en">
	<head>
		<meta charset="UTF-8" />
		<meta http-equiv="X-UA-Compatible" content="IE=edge" />
		<meta name="viewport" content="width=device-width, initial-scale=1.0" />
		<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/flatifycss/dist/css/flatify.min.css" />
		<title>Add button</title>
		<style>
			.add-button{
				margin: 0 5px !important;
			}
		</style>
	</head>
	<body>
        <div class="flex-center">
			<button class="add-button size-xs" aria-label="Add"></button>
			<button class="add-button size-lg" aria-label="Add"></button>
			<button class="add-button size-2x" aria-label="Add"></button>
			<button class="add-button size-3x" aria-label="Add"></button>
		</div>
	</body>
</html>
'
/>

```html title="index.html"
<button class="add-button size-xs" aria-label="Add"></button>
<button class="add-button size-lg" aria-label="Add"></button>
<button class="add-button size-2x" aria-label="Add"></button>
<button class="add-button size-3x" aria-label="Add"></button>
```
