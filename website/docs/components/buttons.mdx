Buttons play a big role in any user interface as they give users the ability
to interact with the website, so we should use them carefully.
It is necessary to know the good and bad practices for a perfect button.
First, take a look at [button cheat sheet](https://www.buttoncheatsheet.com/).

### A simple button

A button with Flatify CSS style can be defined in several ways as mentioned in the button cheatsheet,
for any type of element, you can use the `.button` class but elements like `<button>` or input with button type do not need the class.

<PreviewBox
	height="100px"
	children='
<!DOCTYPE html>
<html lang="en">
	<head>
		<meta charset="UTF-8" />
		<meta http-equiv="X-UA-Compatible" content="IE=edge" />
		<meta name="viewport" content="width=device-width, initial-scale=1.0" />
		<link rel="stylesheet" href="https://raw.githack.com/amir2mi/FlatifyCSS/master/dist/css/flatify-min.css" />
		<title>Buttons</title>
	</head>
	<body>
        <div class="flex-center">
			<button>
				Click Here
			</button>
		</div>
	</body>
</html>
'
/>

```html title="index.html"
<button>Click Here</button>
```

:::info
If you do not want to have Flatify CSS style on elements like `<button>` there is a class called
`.default` that do not allow the button to have Flatify CSS button style primarily.

### Sizes

To change the button size use [size setter classes](/docs/helpers/size-setters).
This classes just set a `font-size` property so it is possible to change it yourself by writing custom CSS.

	children='

### Customization

Each button has these CSS variables for customization, they can be set as `:root` for all buttons default style
of just for an specific selector.

`--flatify__button-bg-color` Button background color  
`--flatify__button-bg-color__focus` Button background color for border and focus  
`--flatify__button-text-color` Button text color  
`--flatify__button-text-color__focus` Button text color for focus

<PreviewBox
	height="150px"
	children='
<!DOCTYPE html>
<html lang="en">
	<head>
		<meta charset="UTF-8" />
		<meta http-equiv="X-UA-Compatible" content="IE=edge" />
		<meta name="viewport" content="width=device-width, initial-scale=1.0" />
		<link rel="stylesheet" href="https://raw.githack.com/amir2mi/FlatifyCSS/master/dist/css/flatify-min.css" />
		<title>Buttons</title>
		<style>
			.my-button-1{
				--flatify__button-bg-color: linear-gradient(to right, #4A00E0, #8E2DE2);
				--flatify__button-bg-color__focus: linear-gradient(to right, #4A00E0, #8E2DE2);
				--flatify__button-text-color: white;
				--flatify__button-text-color__focus: white;
			}
			.my-button-2{
				--flatify__button-bg-color: #48CFAD;
				--flatify__button-bg-color__focus: #37BC9B;
				--flatify__button-text-color: #fff;
				--flatify__button-text-color__focus: #F5F7FA;
			}
		</style>
    </head>
    <body>
        <div class="flex-column-center">
    		<button class="my-button-1">Click Here</button>
    		<button class="my-button-2">Click Here</button>
    	</div>
    </body>

</html>
'
/>

```html title="index.html"
<button class="my-button-1">Click Here</button> <button class="my-button-2">Click Here</button>
```

```css title="style.css"
.my-button-1 {
	--flatify__button-bg-color: linear-gradient(to right, #4a00e0, #8e2de2);
	--flatify__button-bg-color__focus: linear-gradient(to right, #4a00e0, #8e2de2);
	--flatify__button-text-color: white;
	--flatify__button-text-color__focus: white;
}
.my-button-2 {
	--flatify__button-bg-color: #48cfad;
	--flatify__button-bg-color__focus: #37bc9b;
	--flatify__button-text-color: #fff;
	--flatify__button-text-color__focus: #f5f7fa;
}
```
