Buttons play a big role in any user interface as they give users the ability to interact with the website, so we should use them carefully.  
It is necessary to know the good and bad practices for a perfect button.
First, take a look at [button cheat sheet](https://www.buttoncheatsheet.com/).

### A simple button

A button with Flatify CSS style can be defined in several ways as mentioned in the button cheatsheet, for any type of element, you can use the `.button` class, the element will have button style with the active/focus-visible states if it supports.

<PreviewBox
	height="110px"
	children='
<!DOCTYPE html>
<html lang="en">
	<head>
		<meta charset="UTF-8" />
		<meta http-equiv="X-UA-Compatible" content="IE=edge" />
		<meta name="viewport" content="width=device-width, initial-scale=1.0" />
		<link rel="stylesheet" href="https://raw.githack.com/amir2mi/FlatifyCSS/master/dist/css/flatify-min.css" />
		<title>Buttons</title>
	</head>
	<body>
        <div class="flex-center">
			<button class="button">
				Click Here
			</button>
		</div>
	</body>
</html>
'
/>

```html title="index.html"
<button class="button">Click Here</button>
```

### Sizes

To change the button size use [size setter classes](/docs/helpers/size-setters).
This classes just set a `font-size` property so it is possible to change it yourself by writing custom CSS.

<PreviewBox
	height="350px"
	children='
<!DOCTYPE html>
<html lang="en">
	<head>
		<meta charset="UTF-8" />
		<meta http-equiv="X-UA-Compatible" content="IE=edge" />
		<meta name="viewport" content="width=device-width, initial-scale=1.0" />
		<link rel="stylesheet" href="https://raw.githack.com/amir2mi/FlatifyCSS/master/dist/css/flatify-min.css" />
		<title>Buttons</title>
		<style>
			@media screen and (max-width: 650px){
    			.flex-center button {
    				width: 45% !important;
    			}
    		}
		</style>
	</head>
	<body>
        <div class="flex-column-center">
			<button class="button size-sm">Click Here</button>
			<button class="button size-md">Click Here</button>
			<button class="button size-lg">Click Here</button>
			<button class="button size-2x">Click Here</button>
		</div>
	</body>
</html>
'
/>

```html title="index.html"
<button class="button size-sm">Click Here</button>
<button class="button size-md">Click Here</button>
<button class="button size-lg">Click Here</button>
<button class="button size-2x">Click Here</button>
```

### Disabled & active

To make a button disabled you can add the `.disabled` class or HTML `disabled` attribute, also to keep the button in pressed style use the `.active` class.

<PreviewBox
	height="240px"
	children='
<!DOCTYPE html>
<html lang="en">
	<head>
		<meta charset="UTF-8" />
		<meta http-equiv="X-UA-Compatible" content="IE=edge" />
		<meta name="viewport" content="width=device-width, initial-scale=1.0" />
		<link rel="stylesheet" href="https://raw.githack.com/amir2mi/FlatifyCSS/master/dist/css/flatify-min.css" />
		<title>Buttons</title>
		<style>
			@media screen and (max-width: 650px){
    			.flex-center button {
    				width: 45% !important;
    			}
    		}
		</style>
	</head>
	<body>
        <div class="flex-center" style="flex-wrap: wrap">
			<button style="width: 200px; margin: 10px" class="button style-accent disabled">Click Here</button>
			<button style="width: 200px; margin: 10px" class="button style-purple" disabled>Click Here</button>
			<button style="width: 420px; margin: 10px 0" class="button style-green">Click Here</button>

### Customization

Each button has these CSS variables for customization, they can be set as `:root` for all buttons default style or just for an specific selector.

`--flatify__button-bg-color` Button background color  
`--flatify__button-txt-color` Button text color  
`--flatify__button-bg-color__focus` Button background color for border and focus  
`--flatify__button-txt-color__focus` Button text color for focus

<PreviewBox
	height="330px"
	children='
<!DOCTYPE html>
<html lang="en">
	<head>
		<meta charset="UTF-8" />
		<meta http-equiv="X-UA-Compatible" content="IE=edge" />

		<link rel="stylesheet" href="https://raw.githack.com/amir2mi/FlatifyCSS/master/dist/css/flatify-min.css" />
		<title>Buttons</title>
		<style>
			.my-button-1{
				margin: 0.5em !important;
				--flatify__button-bg-color: linear-gradient(15deg, #03001e, #7303c0, #ec38bc, #fdeff9);
				--flatify__button-txt-color: white;
				--flatify__button-bg-color__focus: linear-gradient(25deg, #03001e, #7303c0, #ec38bc, #fdeff9);
				--flatify__button-txt-color__focus: white;
			}
			.my-button-2{
				margin: 0.5em !important;
				--flatify__button-bg-color: #48CFAD;
				--flatify__button-txt-color: #fff;

			}
			.my-button-3{
				margin: 0.5em !important;
				--flatify__button-bg-color: #EC87C0;
				--flatify__button-txt-color: #d770ad;
				--flatify__button-bg-color__focus: #D770AD;
				--flatify__button-txt-color__focus: #fff;
			}
		</style>
    </head>
    <body>
        <div class="flex-column-center">
    		<button class="my-button-1 button size-lg">Custom button</button>
    		<button class="my-button-2 button size-lg">Custom button</button>
    		<button class="my-button-3 button outline size-lg">Custom button</button>
    	</div>
    </body>

</html>
'
/>
