Menu is a handy components, by default [dropdown](/docs/components/dropdown) and [navigation bar](/docs/components/navbar) use this component and style it based on their usage.  
Generally there is a `.menu-items-wrapper` parent element with `.menu-item` children.

:::info Active & disabled states
With `.active` and `.disabled` classes you can show active item of the list and disable a link.
:::
