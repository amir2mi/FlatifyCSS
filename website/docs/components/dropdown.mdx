Dropdowns typically provide a list of items to interact with, however it is also possible to have a complex UI inside dropdowns.

## A simple dropdown

To create a dropdown we need a [button](docs/components/button), [menu items](docs/components/menu), and also an [arrow button](docs/components/arrow-button), these are basics and they should have some special classes to work as expected.
