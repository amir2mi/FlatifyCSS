Here is the list of practical helper classes to style elements,
for example a button, table item, div, etc.  
What they do is pretty simple:
set background, color, and border based on the color palette.

### Main colors

<ColorIndicator color="#1cb0f6" title=".style-accent" inline={true} /> Brings the UI primary color.
