Alignment in CSS is a dilemma for beginners, there are tons of ways to achieve it
but helpers classes are life savers.

## Flex

Flexbox is a great feature in CSS you can have what you need without any effort though,
with vendor prefixes it can be messy if you repeat the main logic all the time,
it is better to use the helper classes.
There are two types of helper classes for vertical and horizontal centering with Flexbox.

:::info
For alignment with flexbox, **the helper class should be applied to the wrapper element** so the child itself
without flex parent do nothing.
:::

`.inline-flex` Treat as inline flex.

### Row

`.flex-center` [row & column] Alignment for both X and Y axes.  
`.flex-center-x` [row] Alignment for X axis.  
`.flex-center-y` [row] Alignment for Y axis.
