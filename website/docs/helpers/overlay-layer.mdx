It happens that you might want to have a layer that comes over the element content, it can be done with CSS `::before` pseudo-element, however, it is a common use case, creating this layer each time is not a logical choice.  
FlatifyCSS gives you a helper class called `.overlay-layer`, it does the same thing and you can just customize it easily without writing boilerplates.

Overlay layers have couple of CSS variables to make them more flexible, here they are:

`--flatify__overlay-layer-priority` `z-index` property of the overlay layer, default is 10  
`--flatify__overlay-layer-opacity` Default is 1 which means 100%  
`--flatify__overlay-blending-mode` Default is normal  
`--flatify__overlay-layer-bg-color` By default inherits darker background color

In the example below we have a box with an image inside it, basically we need and overlay layer to change image color with [`mix-blend-mode` CSS property](https://developer.mozilla.org/en-US/docs/Web/CSS/mix-blend-mode).
