:::info
Note that these classes can be customized in `_config.scss` file (excepts sharp, circle and oval),
there is a variable called `$BORDER_RADIUS_LIST`,
These classes are generated from this list.
So it is possible to create customized helper classes for roundness simply!
:::

```scss {8-10} title="_config.scss"
// This list will be converted to helper classes.
// .edge-[]
$BORDER_RADIUS_LIST: (
	round-xs: 0.38em,
	round-sm: 0.5em,
	round-md: 0.75em,
	round-lg: 1em,
	round-pillow: 88% 12% 89% 11% / 12% 89% 11% 88%,
	round-rice: 100% 0% 100% 0% / 100% 0% 100% 0%,
	round-flower: 1em 0.25em 1em 0.25em,
);
```
