Buttons

Buttons play a big role in any user interface as they give users the ability to interact with the website, so we should use them carefully. It is necessary to know the good and bad practices for a perfect button. First, take a look at button cheat sheet.

A simple buttonâ€‹

A button with Flatify CSS style can be defined in several ways as mentioned in the button cheatsheet, for any type of element, you can use the .button class but elements like <button> or input with button type do not need the class.

index.html
<button>Click Here</button>
info

If you do not want to have Flatify CSS style on elements like <button> there is a class called .default that do not allow the button to have Flatify CSS button style primarily.

index.html
<button class="default">Click Here</button>

Colorsâ€‹

There are helper classes for styling elements like button, first color setter classes for having button which diffrent background and text color.

index.html
<button class="style-accent">Click Here</button>
<button class="style-purple">Click Here</button>
<button class="style-yellow">Click Here</button>
<button class="style-warning">Click Here</button>
<button class="style-danger">Click Here</button>
<button class="style-dark">Click Here</button>

Sizesâ€‹

To change the button size use size setter classes. This classes just set a font-size property so it is possible to change it yourself by writing custom CSS.

index.html
<button class="size-sm">Click Here</button>
<button class="size-md">Click Here</button>
<button class="size-lg">Click Here</button>
<button class="size-2x">Click Here</button>

Borderedâ€‹

By default Flatify CSS buttons does not have border however if you add .bordered class to the button it will have border.

index.html
<button class="style-accent bordered">Click Here</button>
<button class="style-purple bordered">Click Here</button>
<button class="style-yellow bordered">Click Here</button>
<button class="style-warning bordered">Click Here</button>
<button class="style-danger bordered">Click Here</button>
<button class="style-dark bordered">Click Here</button>

Outlineâ€‹

Buttons with outline style get more transition time because of background and text inverted color for :active & :focus-visible states. In this case text color is border color, so there is no need to add .bordered class for outline buttons.

index.html
<button class="style-accent outline">Click Here</button>
<button class="style-purple outline">Click Here</button>
<button class="style-yellow outline">Click Here</button>
<button class="style-warning outline">Click Here</button>
<button class="style-danger outline">Click Here</button>
<button class="style-dark outline">Click Here</button>

Customizationâ€‹

Each button has these CSS variables for customization, they can be set as :root for all buttons default style of just for an specific selector.

--flatify__button-bg-color Button background color
--flatify__button-text-color Button text color
--flatify__button-bg-color__focus Button background color for border and focus
--flatify__button-text-color__focus Button text color for focus

index.html
<button class="my-button-1 size-lg">Custom button</button>
<button class="my-button-2 size-lg">Custom button</button>
<button class="my-button-3 outline size-lg">Custom button</button>
style.css
.my-button-1 {
    margin: 0.5em !important;
    --flatify__button-bg-color: linear-gradient(15deg, #03001e, #7303c0, #ec38bc, #fdeff9);
    --flatify__button-text-color: white;
    --flatify__button-bg-color__focus: linear-gradient(25deg, #03001e, #7303c0, #ec38bc, #fdeff9);
    --flatify__button-text-color__focus: white;
}

.my-button-2 {
    margin: 0.5em !important;
    --flatify__button-bg-color: #48cfad;
    --flatify__button-text-color: #fff;
    --flatify__button-bg-color__focus: #37bc9b;
    --flatify__button-text-color__focus: #fff;
}

.my-button-3 {
    margin: 0.5em !important;
    --flatify__button-bg-color: #ec87c0;
    --flatify__button-text-color: #f5f7fa;
    --flatify__button-bg-color__focus: #d770ad;
    --flatify__button-text-color__focus: #fff;
}
